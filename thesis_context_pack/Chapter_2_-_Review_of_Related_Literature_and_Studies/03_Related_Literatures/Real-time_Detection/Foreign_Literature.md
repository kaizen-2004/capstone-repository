# Real-time Detection - Foreign Literature

Enhancement of both domain in security and emergency response systems was the by product of latest developments in the area of face detection, recognition and intelligence surveillance systems. To be specific, in Deep Learning approaches particularly Convulutional Neural Networks (CNNs) and other models such as Transformer-based models, is now the modern standard for performance in face recognition (Thaer et al., 2025). The methods mentioned shone at learning hierarchical features and managing complex occlusion patterns, such as eyeglasses, masks, and hands covering the face, which remain significant barrier to accurate recognition in real-world environment (Zeng et al., 2021). It is emphasize in this literature that efficient recognition requires not only sophisticated algorithms like CNN or Transformer-based models but also high-quality, various unique datasets, and careful consideration of real-world environment factors, including lighting, pose, shadow, and facial expressions (Mohammed & AlTuwaijari, 2022; Kumar et al., 2019).
Even with all of these developments, there is a gap in the existing solutions because existing solutions either address security for intruder detection or emergency monitoring for fire and disaster alerts which operates independently and not in a unified manner. In these evidences, we see a gap, a compact, multi-modal system that is capable of concurrent detection of intruder and verified fire alerts for residential or small-scale commercial deployment. Moreover, recent literature has shown an barrier usch as low-light recognition, false alarms, and real-time responsiveness which require careful design of both software and hardware, particularly in resource-constrained environments.
