# Phase 4: Iterative Testing, Adjustment, and System Refinement

Scenario-based tests are run in the condo-like environment such as normal activity, simulated intruders, fire-related scenarios, and nuisance conditions. The system’s alerts, event logs, and response times are recorded for each scenario. Research staff adjust thresholds, fusion rules, and alert behavior after each test cycle and then repeat the tests. The data from different cycles are kept to track progress.
