# Recording and Evaluation Instruments

System log files and daily summary reports - record raw detection decisions, sensor readings, timestamps, and alert outcomes for later analysis.
Observation checklists and scenario scripts - facilitate the consistent conducting of test scenarios.
Timing or benchmarking tools - quantify the time elapsed between event occurrence and alert display.
Questionnaires - gather user ratings of alert clarity, timeliness, usefulness, and related quality attributes through Likert-scale items.
