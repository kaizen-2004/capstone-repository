# User Evaluation (If Applicable)

If human respondents are included, short survey ratings can be summarized using:
- frequency
- percentage
- weighted mean
- standard deviation

If respondents are grouped and the researcher needs to determine whether response patterns differ by group, a **chi-square test of independence** may be used, provided the data satisfy the assumptions for grouped categorical analysis.

If no grouped comparison is needed, descriptive statistics alone are sufficient.
