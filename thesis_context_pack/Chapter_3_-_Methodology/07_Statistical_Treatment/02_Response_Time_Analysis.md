# Response Time Analysis

In every trial, the alert latency is determined as the time difference between the occurrence of the event and the alert presentation or recorded alert creation time.

For repeated trials, the following descriptive measures should be reported:
- mean latency
- median latency
- standard deviation
- minimum latency
- maximum latency

If separate event types are tested, response-time summaries may be reported per scenario group such as authorized entry, unknown entry, smoke-only trigger, and smoke with visual fire confirmation.
